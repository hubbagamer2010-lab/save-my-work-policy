# 🚀 Save My Work Pro

**Never lose your work again.**

Save My Work Pro is a powerful browser extension that automatically saves and restores text entered across websites. Whether you're writing emails, filling forms, or typing messages, your progress is always محفوظ (safe).

---

## ✨ Features

- 💾 **Auto Save**  
  Automatically saves text as you type in real-time.

- 🔄 **Instant Restore**  
  Recover lost input after refresh, crash, or accidental tab close.

- 🌐 **Works Everywhere**  
  Supports most websites, forms, and input fields.

- 🔒 **Privacy First**  
  All data is stored locally on your device. No tracking, no data collection.

- ⚡ **Lightweight & Fast**  
  No heavy scripts, no performance impact.

---

## 🧠 How It Works

Save My Work Pro detects input fields on web pages and securely stores your typed content using the browser’s local storage.  

If something goes wrong (refresh, crash, navigation), your data can be restored instantly.

---

## 🔐 Privacy

- ❌ No personal data collection  
- ❌ No tracking or analytics  
- ❌ No data sent to servers  
- ✅ 100% local storage  

Your data stays on your device — always.

---

## 📦 Installation

1. Download or clone this repository  
2. Open Microsoft Edge  
3. Go to `edge://extensions/`  
4. Enable **Developer Mode**  
5. Click **Load unpacked**  
6. Select the extension folder
